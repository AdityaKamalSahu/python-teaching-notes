Type Conversion aur Type Casting in Python (Hinglish)
1. Type Conversion (Automatic)

Jab Python khud hi ek data type ko dusre data type me convert kar deta hai, usse Type Conversion kehte hain.
 int ----> float 
Example:

a = 10    --->  # int   3
b = 5.5  --->   # float 3.----

c = a + b ------->10 + 5.5 = 15.5


print(c)---->15.5
print(type(c))---->float

Output:
n
15.5
<class 'float'>

Explanation:
Yahan a integer hai aur b float hai. Python ne automatically a ko float me convert kar diya aur result float diya.

. Type Casting (Manual)

Jab programmer khud kisi value ka data type change karta hai, usse Type Casting kehte hain.

Example 1: int()

a = "25"----string 
b= int(a)-----integer 


b = int(a)
print(a)-----25
print(b)-----25
print(type(b))-----integer
print(type(a))-----string 


Output:

25
<class 'int'>

Example 2: float()

a = 10 -----integer

b = float(a)

print(b)----10.0
print(type(b))-----float

Output:

10.0
<class 'float'>

Example 3: str()

a = 100----integer

b = str(a)---->string 

print(b)
print(type(b))

Output:

100
<class 'str'>
Easy Definition for Students

✅ Type Conversion: Python khud data type change kare.
✅ Type Casting: Programmer khud data type change kare.

Real-Life Example
age = input("Enter age: ")   # input always string deta hai
age = int(age)              # type casting

Agar int() na lagao to error aayega, kyunki input string hota hai.

Common Type Casting Functions
Function	Use
int()	Integer me convert
float()	Float me convert
str()	String me convert
bool()	Boolean me convert

# Important Note

⚠️ **Whatever you enter using `input()` is stored as a string by default.**

Example:

```python
age = input("Enter your age: ")

print(type(age))
```

**Output:**

```text
<class 'str'>
```

Even if you enter `20`, Python stores it as `"20"`.

---

# Why Type Casting is Needed with Input?

Suppose:

```python
age = input("Enter your age: ")

print(age + 5)
```

This will give an error because:

* `age` = string
* `5` = integer

Python cannot add a string and an integer.

### Correct Way:

```python
age = int(input("Enter your age: "))

print(age + 5)
```

**Output:**

```text
Enter your age: 20
25
```

Here we used **type casting** (`int()`).

# Different Ways of Type Casting with Input

### Integer Input

```python
num = int(input("Enter a number: "))
```

### Float Input

```python
price = float(input("Enter price: "))
```

### String Input

```python
name = str(input("Enter name: "))
```

*(Actually `input()` already returns string, so `str()` is optional.)*

### Boolean Conversion

```python
value = bool(input("Enter something: "))

print(value)
```

If user enters anything → `True`

If empty string / empty input → `False`

| --------- | ------------------------- |

| `str()`   | Convert to string         |
| `bool()`  | Convert to boolean        |

**One-line summary:**
👉 `input()` always gives a string, so whenever you want to do mathematical operations, use **type casting** (`int()` or `float()`).
